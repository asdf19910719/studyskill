# 外部资料检索规则

## 资料优先级

- P0：官方文档、官方教程、官方源码文档。
- P1：官方博客、权威书籍、标准文档。
- P2：高质量系列博客、深度教程、课程讲义。
- P3：社区问答、论坛、StackOverflow、Reddit。
- P4：短视频和碎片化文章，只作辅助理解，不作主要事实来源。

## 筛选规则

1. 是否来自官方？
2. 是否过时？
3. 是否和当前学习目标相关？
4. 是否只是碎片经验？
5. 是否能补足当前资料缺口？
6. 是否适合作为入门教材？
7. 是否适合作为深入资料？

## 搜索关键词生成规则

为每个知识点生成：

1. 中文关键词。
2. 英文关键词。
3. 官方站点限定关键词。
4. 问题导向关键词。
5. 实战 / 排障关键词。

## 示例

知识点：Android vendor 分区

中文关键词：

```text
Android vendor 分区 Treble VNDK
Android system vendor 分区 区别
Android Treble 架构
```

英文关键词：

```text
Android vendor partition Project Treble
Android VNDK overview
Android system vendor partition architecture
```

官方限定：

```text
site:source.android.com Android vendor partition
site:source.android.com VNDK overview
site:source.android.com Project Treble partitions
```
